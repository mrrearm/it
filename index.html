🌐 Indirizzi IP e Configurazione di Rete

📍 Indirizzo IP

Definizione: Identificatore univoco del dispositivo nella rete

192.168.1.100

32 bit divisi in 4 ottetti, identifica l'host all'interno della rete

🎭 Subnet Mask

Definizione: Definisce la porzione di rete e host dell'IP

255.255.255.0

Separa l'indirizzo di rete dall'indirizzo dell'host, determina la dimensione della subnet

🚪 Gateway

Definizione: Punto di accesso verso altre reti

192.168.1.1

Router che connette la rete locale a Internet, gestisce il traffico inter-rete

🔍 DNS

Definizione: Traduce nomi di dominio in indirizzi IP

8.8.8.8

Risolve www.google.com in 172.217.16.164, essenziale per la navigazione web

🔗 Differenze Chiave:

IP: Identifica IL dispositivo | Subnet Mask: Definisce LA rete | Gateway: Collega LE reti | DNS: Traduce I nomi

💾 Strategie di Backup

🔴 Backup Completo (Full)

Caratteristiche: Copia TUTTI i file e cartelle

Vantaggi: Ripristino veloce, copia completa

Svantaggi: Richiede più tempo e spazio

Frequenza: Settimanale o mensile

Lunedì: 100GB di dati

🟡 Backup Incrementale

Caratteristiche: Copia solo i file MODIFICATI dall'ultimo backup

Vantaggi: Veloce, occupa poco spazio

Svantaggi: Ripristino più lento (serve catena completa)

Frequenza: Giornaliera

Martedì: +2GB, Mercoledì: +1.5GB

🟢 Backup Differenziale

Caratteristiche: Copia i file modificati dall'ULTIMO backup completo

Vantaggi: Ripristino più veloce dell'incrementale

Svantaggi: Dimensioni crescenti nel tiempo

Frequenza: Giornaliera o ogni 2-3 giorni

Martedì: 2GB, Mercoledì: 3.5GB

📊 Confronto Rapido:

Completo: Tutto sempre | Incrementale: Solo le novità | Differenziale: Tutto dal completo

👤 Creazione Utenti in Windows

⚙️ Pannello di Controllo

Account Utente → Gestisci un altro account → Aggiungi nuovo utente

💻 Impostazioni Windows

Impostazioni → Account → Famiglia e altri utenti → Aggiungi un altro utente

🔧 Gestione Computer

Utenti e gruppi locali → Utenti → Nuovo utente (click destro)

⌨️ Command Prompt

cmd → net user [nome] [password] /add

🔐 Tipi di Account:

Administrator

Controllo completo del sistema, installazione software, modifica impostazioni critiche

Standard User

Utilizzo normale, non può installare software o modificare impostazioni di sistema

Guest

Accesso limitato e temporaneo, nessuna personalizzazione permanente

💻 Installazione Windows 10

Preparazione Media

Scaricare Windows 10 Media Creation Tool, creare USB bootable (almeno 8GB)

Configurazione BIOS/UEFI

Accedere al BIOS, impostare boot priority su USB, abilitare UEFI se necessario

Avvio Installazione

Boot da USB, selezionare lingua, formato ora e valuta, layout tastiera

Partizionamento

Selezionare disco, creare/eliminare partizioni, formattare in NTFS

Installazione Sistema

Copia file, installazione funzionalità, riavvii automatici (20-45 minuti)

Configurazione Iniziale

Account Microsoft/locale, PIN, privacy, Cortana, aggiornamenti

📊 Installazione Microsoft Office

🌐 Abbonamento Office 365

Login su office.com → Installa Office → Scarica ed esegui installer

Download

Esecuzione automatica, installazione delle applicazioni selezionate

Attivazione

Login con account Microsoft, inserimento product key se richiesto

Configurazione

Primo avvio applicazioni, configurazione tema, sincronizzazione impostazioni

🚨 Procedura di Emergenza:

Boot da USB Recovery

Creare disco di ripristino, boot da USB, accesso a strumenti ripristino

Ripristino Automatico

Windows tenta riparazione automatica, diagnosi problemi avvio

Ripristino Manuale

Prompt comandi, sfc /scannow, chkdsk, bootrec /fixboot

⚙️ Configurazione Rete Domestica

🌐 Router Setup

192.168.1.1 → Admin panel → Configurazione guidata → SSID e Password

🔒 Sicurezza WiFi

WPA3/WPA2 → Password complessa → Disabilitare WPS → Guest Network

📱 Connessione Dispositivi

WiFi Settings → Seleziona rete → Inserisci password → Connetti automaticamente

🖥️ IP Statico

Proprietà scheda → TCP/IPv4 → Usa indirizzo IP specifico

🔧 Configurazione Tipica Rete Domestica:

Router IP: 192.168.1.1 | Subnet: 255.255.255.0 | DHCP Range: 192.168.1.2-254

DNS Primario: 8.8.8.8 | DNS Secondario: 8.8.4.4

🔍 Risoluzione Problemi:

ipconfig /release /renew

Rilascia e rinnova indirizzo IP automatico

ping 8.8.8.8

Testa connettività Internet

nslookup google.com

Verifica funzionamento DNS
